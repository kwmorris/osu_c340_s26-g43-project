# OSU CS340 S26 - Group 43 Project

## Install instructions:

Clone the repository.
Open a terminal instance with the root folder open.  
Run "npm install" in the terminal.  
Update model.mjs with the correct DB host, login, & table.

### Running a DEV instance
Run "npm run dev" in the terminal.

### Running a PROD instance
Run "npm run build" in the terminal.  
Run "npm run production" in the terminal.

### Updating a PROD instance
Run "npm run build" in the terminal.

### Stopping a PROD instance
Run "npm run stop_production" in the terminal.
